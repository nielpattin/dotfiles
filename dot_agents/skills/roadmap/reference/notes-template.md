## Auto-Updates

This file is automatically updated by:

| Command | Updates |
|---------|---------|
| `/complete-next-task` (first task) | Adds **Started** date |
| `/complete-next-task` (all done) | Adds **Completed** date |
| Manual | Add learnings, gotchas, decisions |

---

## Usage Tips

1. **After each feature**: Add patterns discovered to "Learnings"
2. **When stuck**: Document gotchas for future reference
3. **On decisions**: Record why you chose an approach
4. **Weekly review**: Update notes with accumulated learnings
